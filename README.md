# 🎮 Tic Tac Toe Game

## 📌 Description

This is a simple **Tic Tac Toe game** built using **HTML, CSS, and JavaScript**.
It is a mini project that demonstrates basic front-end development skills, including DOM manipulation, event handling, and game logic implementation.

---

## 🚀 Features

* 🧑‍🤝‍🧑 Two-player mode (Player X vs Player O)
* 🎯 Real-time game result detection (Win / Draw)
* 🔄 Reset / Restart game functionality
* 🎨 Clean and responsive user interface
* ⚡ Fast and interactive gameplay

---

## 🛠️ Technologies Used

* **HTML** – Structure of the game
* **CSS** – Styling and layout
* **JavaScript** – Game logic and interactivity

## 🎮 How to Play

* The game is played on a 3×3 grid
* Player 1 uses **X**, Player 2 uses **O**
* Players take turns marking empty cells
* The first player to align 3 symbols (row, column, or diagonal) wins
* If all cells are filled without a winner → it's a **draw**
